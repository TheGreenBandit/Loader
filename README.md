A loader made in dx11, using imgui to display.

based upon [Niplix's dx9 loader](https://github.com/ALEHACKsp/Loader-Base/tree/main) with heavy changes

to build, make sure you have cmake installed. Then open command prompt, cd into the directory, mkdir build if you havent and cd into that, then run cmake ..
To use the chat feature you will have to create your own discord application and bot then edit the token in token.hpp accordingly.
